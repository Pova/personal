# Disc Golf Data

End-to-end data project for exploring what makes Gannon Buhr elite against top MPO fields.

The project has two parts:

- `pipeline/`: Python extraction, parsing, Postgres loading, transformation, and static JSON export.
- `src/`: SvelteKit + D3 data story consuming generated JSON from `static/data/`.

## Pipeline

```powershell
cd disc-golf/pipeline
python -m pip install -e .
python -m dgdata.cli audit
python -m dgdata.cli export
```

To load the seed dataset into Postgres:

```powershell
$env:DATABASE_URL = "postgresql://user:password@localhost:5432/disc_golf"
python -m dgdata.cli schema
python -m dgdata.cli load-seed
```

The checked-in seed dataset is for pipeline and visualization development. Replace it with scraped source data before publishing final analytical claims.

## Website

Requires Node `>=20.19.0`.

```powershell
cd disc-golf
npm install
npm run dev
```

The Svelte app reads `static/data/gannon-buhr-analysis.json`, which is generated by the Python export command.
