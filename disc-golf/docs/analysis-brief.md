# Analysis Brief

## V1 Question

What makes Gannon Buhr elite compared with other top MPO players?

The first version answers this through scoring outcomes, consistency, ceiling, and performance in elite events. It does not make unsupported claims about putting, driving, or shot selection unless those fields are later sourced.

## Comparison Cohort

The initial cohort is intentionally small so the pipeline can be validated end to end:

- Gannon Buhr
- Isaac Robinson
- Calvin Heimburg
- Ricky Wysocki
- Eagle McMahon
- Paul McBeth
- Anthony Barela
- Kyle Klein

The cohort can expand once source coverage and parser quality are proven.

## Event Window

Start with MPO elite events and majors from 2021 onward. This captures Gannon's rise while keeping the source audit tractable.

## Claims To Support

- Win, podium, top-5, and top-10 rates.
- Average finish and finish percentile.
- Round rating distribution, ceiling, and variance.
- Field-adjusted scoring margin.
- Final-round movement and lead protection where round data is available.
- Year-over-year progression.

## Caveats

This project should describe V1 as a scoring-profile analysis. True skill decomposition requires granular stats that may not be publicly available across the full time window.
